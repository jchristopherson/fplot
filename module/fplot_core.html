FPLOT is a Fortran library providing a means of interacting with GNUPLOT from a Fortran program. The library is designed in an object-oriented manner, and as such utilizes language features that require a compiler that supports the 2003 standard.
Additionally, it is expected that Gnuplot is installed on the system path. For full functionallity, a minimum of GNUPLOT v5.2 is expected.
